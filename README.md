# Two power
## It prints powers of 4 and 5 up to the number entered.
### An example
![An screenshot of the program](./ss-example.png)

### Requirements:
* JDK (version 17 is recommended)

### To run:
Open the terminal on the project directory and type:

```
$ cd src
$ java Main.java
```
